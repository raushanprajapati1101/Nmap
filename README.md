# 🔍 Python Nmap Automation Script

This project is a Python-based wrapper for **Nmap** that allows users to perform network scans in a simple and interactive way.

## 🚀 Features

* Easy-to-use interface for running Nmap scans
* Supports multiple scan options
* Beginner-friendly workflow
* Automates common reconnaissance tasks

## ⚙️ Requirements

Before running the script, make sure you have the following installed:

* Python 3.x
* Nmap
* `python-nmap` library

Install the required Python library:

```bash
pip install python-nmap
```

## 🛠️ Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

2. Run the script:

```bash
python3 script_name.py
```

3. Follow the on-screen prompts to select scan options.

## 📌 Notes

* Ensure Nmap is installed and added to your system PATH.
* Some scans may require root/administrator privileges.

## 📖 About

This project is created for learning and practicing network reconnaissance using Python and Nmap.
